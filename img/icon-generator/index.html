使用说明

支持的格式

  • ICO: Windows 系统图标格式，支持多尺寸
  • ICNS: macOS 系统图标格式，支持 Retina 显示

推荐尺寸

  • Windows ICO: 16×16, 32×32, 48×48, 64×64, 128×128, 256×256
  • macOS ICNS: 16×16, 32×32, 64×64, 128×128, 256×256, 512×512, 1024×1024

使用建议

  • 建议使用正方形、高分辨率的原图
  • 图标设计应简洁明了，避免过多细节
  • 透明背景适合大多数应用场景
  • 生成的图标可直接用于软件开发
